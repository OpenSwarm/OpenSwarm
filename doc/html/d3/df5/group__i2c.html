Detailed Description

Functions to read from and write on the I2C interface.

Author
Stefan M. Trenkwalder s.tre.nosp@m.nkwa.nosp@m.lder@.nosp@m.open.nosp@m.swarm.nosp@m..org

Inter-Integrated Circuit bus is a multi-master, multi-slave, serial bus (see also https://en.wikipedia.org/wiki/I%C2%B2C )

Usage

The I2C interface can be initialised and started with Sys_Init_I2C() and Sys_Start_I2C() respectively. Similarly, it can be paused, continued, or stopped by Sys_Pause_I2C(), Sys_Contine_I2C(), or Sys_Stop_I2C() respectively. While the interface is running, data can be written with Sys_I2C_SentBytes(uint8, uint8 *, uint16). Values can be read with Sys_I2C_Read(uint8 , uint8 *, uint16, pByteFunction) where the request message has also to be specified.
